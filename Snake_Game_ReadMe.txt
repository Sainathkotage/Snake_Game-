# 🐍 Snake Game in Python (Pygame)

A simple classic **Snake Game** built using **Python and Pygame**.  
Control the snake, eat the food, grow longer — but don’t run into the walls or yourself!

## 🎮 Features
- Real-time keyboard controls (arrow keys)
- Score tracking
- Random food generation
- Game over and restart options
- Clean and beginner-friendly code

## 🛠️ Technologies
- Python 3
- Pygame

## 🚀 How to Run

1. Install `pygame`:
   ```bash
   pip install pygame
   ```

2. Run the game:
   ```bash
   python snake_game.py
   ```

## 💡 Perfect for:
- Python beginners
- Learning game loops and event handling
- Practicing logic and conditionals
